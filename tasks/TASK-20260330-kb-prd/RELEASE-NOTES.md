# RELEASE NOTES

## 公司内部知识库 PRD v0.1

### 本次产出
- 新增《公司内部知识库 PRD 初版》正式文档
- 文档状态：review 已通过，进入正式发布阶段
- 发布范围：GitHub 工作仓库 + Notion（若外部权限/认证齐备）

### 核心内容摘要
1. 明确产品定位为企业内部知识资产管理平台。
2. 补齐异地登录、安全控制、权限分级、可溯源、知识生命周期等核心章节。
3. 明确“访客”仅指内部受限访客，不扩展为外部匿名访问。
4. 补齐账号与权限生命周期规则：入职、转岗、离职/停用、权限回收、知识资产归属、审计保留。
5. 给出角色-权限矩阵、文档分级规则、搜索与检索边界、审计字段要求。
6. 增补待确认项处理规则，区分可选项与实施阻塞前提。

### 建议 review 重点
- 业务范围是否收敛合理
- 权限模型与文档分级是否满足管理要求
- 异地登录控制口径是否符合企业安全预期
- 审计留痕字段与日志保留要求是否充分
- 待确认项划分是否清晰
